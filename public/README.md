# Serpentine Galleries GitHub
This is the public GitHub organisation account for Serpentine Galleries:
https://github.com/serpentinegalleries

We use this space to host and share code-based projects developed through our Arts Technologies programme and other initiatives. Projects include prototypes, research tools, software, and collaborative commissions.
